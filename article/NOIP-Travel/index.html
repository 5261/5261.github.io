NOIP 游记

NOIP 2016，早就期待而恐惧的时刻啊，就这样到了。

Day -1

上午打了一上午的板子，模板题各种 1A 还是资磁滴~

中午颓了一中午 KOF ，感觉手更累了...

下午不知该干什么，然后一下午就过去了，快晚上时打了人生中第一局 Deemo，41%，果真手残...

NOIP Day0 前夜，整个机房充满了愉(tui)快(fei)的气息...

Day 0

出发前

准备了纸、笔，随便热了会手（这时候热手有什么用啊喂）

上午十点准备出发。

按 OI 传统，此时也许该说：

RP++, Bless All.

车上

座位 正好不够，然后被挤在角落，然后左边的人在搓炉石，右边的人在用 手机 打 Pascal 的 Kruskal 代码，前面的小伙伴们在玩 狼人杀。

我？发呆胡想 + 看狼人杀 ...

到啦

到了日照一中，报道。

抽签，要去 曲阜师范大学 考，可那是什么地方...

回宿舍，里面很干净。

试机

Day0 晚上要试机啦，没想到曲师大这么近，坐上车就到了。

Windows XP 环境，DevC++ 正常，gdb 和 g++ 的版本好新啊，有 Python2，可为什么会出现 Eclipse 和 VS2008 啊，简直不敢相信。

监考人员讲注意事项，不允许带零食饮料差评！

照旧检查键盘，硬盘保护情况，打了会儿代码试机就结束啦，感觉过的好快。

Day1

电子格式题目，但解压密码毫无意义而且很难输入差评...

总览题目，T1 是个正常的 D1T1，T2 看上去很可做，T3 期望是什么鬼啦。

T1 很快写完，Mogician 真是 exciting。

T2 先打暴力和部分分，然后试图想正解（真是作死），想了好几种方式对路径分类，但都不对，时间很快就过去了...

T3 废了点儿劲才读懂题目，然后打暴力，调了好久。

整理文件，交题。

总的来说时间主要花在 T2 上了，留给 T3 的时间不够，而且因为对期望很陌生，所以做 T3 时完全慌了。

说不紧张，还是紧张了。

谁让这是第一次，也是唯一一次机会呢？

考试结束回来，大家几乎都表示 Day1 爆炸。

心情有些低落，强行振作，还有 Day2 呢。

Day2

还没解压看到压缩包的里 angrybirds 就感到一阵阵的恐惧...

T1 数学题，组合数写了质因数分解，感觉并跑不过，两种暴力对拍了一下正确性应该没有问题？

T2 平衡树应该可做？但怎么敢在 NOIP 考场上写平衡树啊喂，于是强行 set 拿部分分，写完发现可以用优先队列，但没改算了，和暴力对拍比较顺利。

T3 n 很小状压可做？想到两只猪和出发点可以确定抛物线，所以就愉快地开写啦。

一直调不过样例，思路应该没有问题呀，到结束也没想出来哪里错了。

整理文件，交题。

NOIP 结束，反正一切都尘埃落定，什么也不想想了。

当晚就回学校了，感觉并没落下多少文化课。

后续

程序出来，自测。

XJ 数据 361 分，Luogu 数据 287 分，感觉一等无望。

无论如何，生活继续，该奋斗的还是要奋斗呀~

还是很想早些知道成绩啊。
